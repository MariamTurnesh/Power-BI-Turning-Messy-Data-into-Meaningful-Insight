## Project Focus

Real-world data is rarely analysis-ready. This project demonstrates how Power BI is used to:

- Clean and transform messy data using **Power Query**
- Model data using **fact and dimension tables**
- Build correct **one-to-many relationships**
- Create meaningful **DAX measures**
- Design dashboards that respond correctly to slicers and filters

The emphasis is on **data preparation, modeling, and analytical thinking**, not just visuals.
